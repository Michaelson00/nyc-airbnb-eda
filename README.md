NYC Airbnb Data Cleaning & Exploratory Data Analysis

Project Overview

This project focuses on analyzing the New York City Airbnb dataset to uncover trends in pricing, reviews, and neighborhood influence. The main goal was to clean and wrangle the data, then perform an exploratory data analysis to understand patterns across room types and boroughs. Key objectives included investigating the pricing structure, evaluating the relationship between reviews and price, and exploring the impact of neighborhood location on listings. The analysis provides insights for hosts and travelers alike.

Key Findings

1. Minimal price variation across different room types.

2. Weak differentiation in prices between boroughs.

3. Near-zero correlation between listing price and number of reviews.
